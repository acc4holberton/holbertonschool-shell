Add 0-iam_betty script to change user to betty
- 1-who_am_i: prints the effective username of the current user
- 2-groups: prints all groups the current user is part of
- 3-new_owner: changes the owner of the file hello to betty
- 4-empty: creates an empty file called hello
- 5-execute: adds execute permission to the owner of the file hello
- 6-multiple_permissions: adds execute permission to owner and group, read permission to others for hello
- 7-everybody: adds execution permission to owner, group, and others for hello
- 8-James_Bond: sets permissions of hello to none for owner/group and all for others
- 9-John_Doe: sets permissions of hello to rwx for owner, r-x for group, and -wx for others
- 10-mirror_permissions: sets the mode of hello to be the same as olleh
- 11-directories_permissions: adds execute permission to all subdirectories for owner, group, and others
- 12-directory_permissions: creates my_dir with permissions 751 in the working directory
- 13-change_group: changes the group owner of hello to school
- 14-change_owner_and_group: changes owner to vincent and group to staff for all files and directories in the working directory
- 15-symbolic_link_permissions: changes owner and group of _hello symbolic link to vincent and staff
